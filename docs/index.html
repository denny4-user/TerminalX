TerminalX
IR-мультитул для M5Stack StickS3
Нужен Chrome или Edge на десктопе (Web Serial). Работает только по HTTPS.
  1. Открой в Chrome или Edge на компьютере.
  2. Подключи M5StickS3 по USB-C.
  3. Нажми Прошить и выбери COM-порт устройства.
  4. Дождись окончания — устройство перезагрузится в TerminalX.
Функции TerminalX предназначены только для тестирования собственных устройств.
